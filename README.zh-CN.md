# RedNote Image Copy Helper / 小红书图片下载与复制助手

这是一个基于 Tampermonkey 的浏览器用户脚本，用于提升在 RedNote / 小红书网页端收集图文素材的效率。

截图之后再补充。

## 项目简介

这个脚本把“小红书笔记文字复制”和“小红书图片保存”两个常用动作整合到一个统一的网页浮动面板中。打开小红书网页端笔记后，可以直接复制标题、作者、发布时间、正文和当前页面链接，也可以下载当前可见大图、下载已加载的大图，或复制图片链接。

## 功能特点

* 识别小红书页面中的大图
* 支持下载当前可见图片
* 支持下载当前页面已加载的全部图片，单次最多 30 张
* 只会下载当前打开的笔记详情页中的图片
* 不会直接下载首页 feed 里所有已加载图片
* 支持复制图片链接
* 支持复制笔记文字，包括标题、作者、发布时间、正文和当前页面 URL
* 复制文字时同时写入 `text/html` 和 `text/plain`，方便粘贴到 Word 或 AI 工具
* 图片文件名优先使用笔记标题，标题为空时再使用 noteId
* 提供可拖动的网页浮动面板
* 支持隐藏 / 显示浮动面板
* 支持中英文切换
* 支持自动跟随小红书浅色 / 深色模式，也可以手动切换主题
* 支持复制 Probe JSON，方便调试小红书页面结构
* 自动记住面板位置和隐藏状态
* 适合内容研究、视觉参考收集和个人资料整理流程

## 使用场景

这个工具主要用于个人效率提升、内容分析和研究型工作流。它可以帮助用户在浏览小红书网页端时，更方便地整理图文信息，减少手动复制、下载和排版的重复操作。

## 安装方式

1. 在浏览器中安装 Tampermonkey。
2. 打开本仓库中的 `rednote-image-copy-helper.user.js` 文件。
3. 点击 GitHub 页面右上方的 `Raw`。
4. Tampermonkey 会自动识别用户脚本并打开安装页面。
5. 点击 `Install` 完成安装。

## 使用方式

1. 打开小红书网页端笔记页面。
2. 页面上会出现一个浮动面板。
3. 你可以使用面板中的按钮：

   * 复制文字
   * 下载当前图
   * 下载全部图
   * 复制图链
   * 复制 Probe

4. 可以拖动面板到页面任意位置。
5. 不需要时可以隐藏面板，需要时再点击迷你按钮显示。
6. 点击「设置」可以切换中文 / English，以及自动 / 浅色 / 深色主题。

图片下载和图链复制只针对当前打开的笔记详情页。如果停留在首页 feed 且没有打开任何笔记详情层，脚本会提示你先打开一篇笔记，不会扫描并下载首页 feed 的图片。
